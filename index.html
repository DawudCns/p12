Refonte de la Plateforme Kasa

Client : Startup spécialisée dans la location d'appartements entre particuliers.
Présentation : J'ai travaillé en tant que développeur front-end sur le projet Kasa, visant une refonte complète du site web, avec une transition de ASP.NET à une stack Javascript moderne. Mon rôle consistait à concevoir une plateforme web en utilisant React pour le front-end et Node.js pour le back-end, en respectant les maquettes fournies par le designer.

Objectifs :
L'objectif était de moderniser la plateforme existante en utilisant React, afin d'optimiser l'expérience utilisateur sur différentes tailles d'écran. J'ai enrichi l'interactivité du site en intégrant des animations CSS et en ajoutant des éléments avancés, comme la création d'un carrousel pour afficher les images des logements, l'implémentation d'un collapse pour améliorer l'UX, et la gestion dynamique des fiches logements à partir de données internes. De plus, j'ai utilisé Sass pour améliorer la gestion des styles.

Compétences Utilisées :
[React, Node.js, Sass, Responsive, Web Design, Figma, Animations CSS]

Amélioration du SEO et Finalisation du Site Web

Présentation :
En tant que développeur front-end, j'ai collaboré avec une équipe pour finaliser le nouveau site internet d'une architecte d'intérieur, dans le cadre d'un projet visant à améliorer le SEO. J'étais responsable du développement de plusieurs fonctionnalités clés pour le site.

Objectifs :
Mon rôle comprenait la création de la page de présentation des travaux de l'architecte, avec des options de filtrage par catégorie, le développement de la page de connexion sécurisée de l'administrateur en utilisant JWT (JSON Web Tokens) pour l'authentification, ainsi que la conception d'une modale permettant à l'architecte d'ajouter ou de supprimer des projets. J'ai utilisé JavaScript pour générer dynamiquement les éléments en communiquant avec une API, ce qui a permis de récupérer, ajouter et supprimer des travaux. Tout au long du projet, nous avons utilisé un tableau Kanban détaillé pour organiser notre travail de manière efficace. Nous avons également optimisé le site pour le SEO en améliorant la structure des balises HTML, en compressant les images et en ajoutant des balises meta pertinentes.

Compétences Utilisées :
[Node.js, Figma, JavaScript, JWT, Swagger, SEO Optimization]
